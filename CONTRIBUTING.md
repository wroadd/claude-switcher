# Contributing

Thank you for helping improve Claude Switcher.

1. Open an issue before substantial behavior or credential-format changes.
2. Create a focused branch from `main`.
3. Add or update tests for privileged-process changes.
4. Run `pnpm check`.
5. Explain security impact and manual verification in the pull request.

Never include real Claude credentials, emails from third parties, authentication output containing identifiers, or application-data snapshots in issues, tests, or commits.
