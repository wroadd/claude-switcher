# Roadmap

## 0.1 — Secure local MVP

- [x] OAuth profile capture
- [x] OS-backed encrypted vault
- [x] macOS Keychain activation
- [x] credentials-file activation
- [x] running-process guard
- [x] backup before activation
- [x] local activity trail

## 0.2 — Platform hardening

- [ ] signed and notarized builds
- [ ] native menu-bar and tray switching
- [ ] verified Windows native Claude Code credential integration
- [ ] Linux secret-service compatibility matrix
- [ ] restore-from-backup UI
- [ ] export/import using a user-supplied passphrase and authenticated encryption

## 0.3 — Expanded workflows

- [ ] safe API-key session launching without plaintext persistence
- [ ] profile health checks and re-authentication guidance
- [ ] optional per-project profile preference
- [ ] CLI companion for scripted activation

Usage/quota scraping is not currently planned because Claude Code does not expose a stable, documented local quota API suitable for this project.
